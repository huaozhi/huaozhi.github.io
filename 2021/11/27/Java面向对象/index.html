可变参数

public int getSum(int... nums) { // int... 表示接受的是可变参数，类型是 int ,即可以接收多个 int(0-多)
    int n = num.length, sum = 0;
    for (int i = 0; i < n; i ++) {
        sum += num[i];
    }
}
  • 可变参数的实参可以是0个或任意多个
  • 可变参数的实参可以为数组
  • 可变参数的实质就是数组
  • 可变参数可以和普通类型的参数一起放在形参列表，但必须保证可变参数在最后
  • 一个形参列表只能出现一个可变参数

包

包的三大作用

  • 区分相同名字的类
  • 方便管理类
  • 控制访问范围

包的命名

  • 只能包含字母、数字、下划线、小圆点
  • 不能包含关键字
  • 不能数字开头
  • 不能是关键字或保留字
  • 一般是小写字母+小圆点
  • 一般是com.公司名.项目名.业务模块名

常用包

  • java.lang.* :基本包，默认引入，不需要再引入.
  • java.util.*: 系统提供的工具包, 工具类，使用 Scanner
  • java.net.*:网络包，网络开发
  • java.awt.*:是做 java 的界面开发，GUI

注意事项

  • package 的作用是声明当前类所在的包，需要放在类(或者文件)的最上面
  • 一个类中最多只有一句 package

访问修饰符

  • public ：本类、同一个包、子类、不同包都可以访问
  • protected ：本类、同一个包、子类可以访问，不同包不可以访问
  • 默认 ：本类、同一个包可以访问，子类、不同包不可以访问
  • private ：只有本类可以访问，同一个包、子类、不同包不可以访问

方法重载与方法重写

名称 发生范围 方法名 形参列表 返回类型 修饰符
重载(overload) 本类 相同 类型、个数或顺序至少一个不一样 无要求 无要求
重写(override) 父子类 相同 类型、个数和顺序都要一样 相同或者是父类返回类型的子类 不能缩小父类的访问范围

封装、继承、多态

多态

多态是建立在封装和继承的基础上的

  • 方法的多态

    • 重载

      参数的类型、个数或顺序不同调用不同的方法

    • 重写

      父类子类调用不同的方法

  • 对象的多态（关键、重难点）

    1. 一个对象的编译类型和运行类型可以不一致，父类对象的引用可以指向子类对象

      Animal animal = new Dog(); animal的编译类型是Animal，运行类型是Dog

    2. 编译类型在编译对象时就已经确定了，不能改变

    3. 运行类型是可以改变的

    4. 编译类型看定义时 = 的左边， 运行类型看 = 的右边

多态的向上转型

  1. 本质：父类的引用指向了子类的对象
  2. 语法：父类类型 引用名 = new 子类类型()；
  3. 特点：
    • 编译类型看左边，运行类型看右边
    • 可以调用父类中的所有成员，但是遵循子类对父类的对应访问权限
    • 不能调用子类的特有成员，因为能调用什么成员在编译阶段已经确定
    • 最终运行效果看子类具体实现，如子类方法重写了父类方法，则实现子类方法

多态的向下转型

  1. 语法：子类类型 引用名 = （子类类型）父类引用
  2. 只能强制转换父类的引用，不能转换父类对象
  3. 要求父类引用必须是指向当前目标类型的对象
  4. 向下转型后，就可以调用子类的所有成员了
