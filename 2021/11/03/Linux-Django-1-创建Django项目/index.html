Linux_Django_1_创建Django项目

2021-11-03 Django框架课 413 字
AcWing课程

准备工作-配置环境

  • docker load -i django_lesson_1_0.tar 将镜像加载到本地

  • docker run -p 20000:22 -p 8000:8000 --name django_server -itd django_lesson:1.0

    创建并运行django_lesson:1.0的镜像的容器（要去云平台打开端口）

  • docker attach django_server进入创建的docker容器

  • passwd设置root密码

  • sudo adduser acs创建普通用户acs

  • sudo usermod -aG sudo acs给用户acs分配sudo权限

  • su acs切换用户

  • ctrl+p ctrl+q挂起容器

  • 配置（容器）服务器别名和免密登录

    • vim config配置别名
    • ssh-copy-id 别名为acs用户添加公钥，免密登录
    • scp .vimrc .tmux.conf 别名:传一下祖传配置

项目开始

  • 打开tmux，进入服务器
  • django-admin startproject acapp创建项目acapp
  • 配置git
    • ssh-keygen生成密钥
    • git init
    • git config --global user.name haz
    • git config --global user.email 1143958705@qq.com
    • git add .
    • git commit -m ""
    • 连接到github，git remote add origin git@github.com:huaozhi/acapp.git
    • git pull origin main --allow-unrelated-histories注意要先拉取GitHub的初始分支main并合并
    • git push
  • 再打开一个tmux，一个用于维护项目运行，一个用于开发
  • python3 manage.py runserver 0.0.0.0:8000运行服务器
  • ag 文件名全文搜索位置
  • python3 manage.py startapp game创建game app
  • 登录django管理员页面
    • ctrl c关掉服务器
    • python3 manage.py migrate同步一下数据库的修改
    • python3 managy.py createsuperuser创建管理员用户
    • python3 manage.py runserver 0.0.0.0:8000启动服务器
    • 地址：8000/admin
  • cd game
  • touch urls.py
  • mkdir templates
  • vim views定义函数，返回html
  • vim urls.py写路由
  • cd .. vim urls.py写入总路由
    • models：存储各种数据结构（class）
    • views：存储函数
    • urls：路由
    • templates：存储模板
